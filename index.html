The problem

Plex has a great Live TV & DVR — but only if it sees a tuner. Getting IPTV in there means wrestling tools like xTeVe or Threadfin: fiddly EPG mapping, streams that buffer and drop, phantom "all tuners busy", configs that corrupt, and Plex's own bug where every airing that shares a title gets the same description. It's powerful, but it's a weekend of yak-shaving.

The fix — Tunaar

📺

HDHomeRun emulation

Answers Silicondust discovery, so Plex/Emby/Jellyfin auto-find it as a network tuner — no manual IP.

🔗

Merge everything

Many IPTV playlists and a real HDHomeRun behind one device, with collision-free numbering.

🗓️

Guide that just works

Auto-detects the XMLTV guide from your playlist and merges extras — with name-matching & manual mapping.

🩹

Plex duplicate-description fix

Keeps a separate description per airing — clean titles, date-based episode numbers, movies left alone.

🎞️

Reliable streaming

ffmpeg-remuxed MPEG-TS with real tuner slots that release the instant a client disconnects.

🎛️

Editable dashboard

Add sources, pick groups, map channels and see live activity — no config files, no SSH.

Up and running in two minutes

Run one container

docker run -d --name tunaar \
  --restart unless-stopped --network host \
  -v tunaar-config:/config \
  ghcr.io/munerisuk/tunaar:latest

Add your playlist

Open http://host:5004, paste your M3U (or pick a free country list), and Tunaar builds the lineup + guide. See the install guide.

Add the tuner in Plex

Plex auto-discovers it. Point the guide at /epg.xml and you've got Live TV & DVR. See the user guide.
